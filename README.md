**ruDocs** (ранее **PS-Commands**) - это большая база заметок по синтаксису [PowerShell](/posh.md), командам [Linux](/linux.md) и инструментам [DevOps](/devops.md) на русском языке, собранные на пути от системного администратора Windows до DevOps инженера.

> [!TIP]
> Вы можете ознакомиться с другими шпаргалками для языков [Go](https://github.com/Lifailon/golang-cheat-sheet-ru) и [JavaScript](https://github.com/Lifailon/node.js-cheat-sheet-ru).

Репозиторий также содержит набор PowerShell [скриптов](/Scripts), коллекцию стеков [Docker Compose](/Docker-Compose) и манифестов [Kubernetes](/Kubernetes), которые поддерживаются в актуальном состояние и эксплуатируется в домашней лаборатории.
